# SoulForge

SoulForge is an advanced, AI-powered terminal IDE that supercharges modern software development with intelligent code navigation, powerful refactoring, and seamless AI assistance.

## Features

- **Intelligent Code Navigation**: Quickly find definitions, references, symbols, imports, and more across your codebase.
- **Powerful Refactoring**: Safely rename, extract, format, and organize code with compiler-guaranteed transformations.
- **Contextual Analysis**: Get diagnostics, type information, outlines, and semantic diffs to understand your code.
- **Test Scaffolding**: Automatically generate test skeletons from your source files.
- **Git Superpowers**: Manage your repository with advanced git commands and visualizations.
- **AI-Driven Development**: Dispatch subagents to research, implement, and coordinate complex tasks.
- **Persistent Memory**: Record decisions, patterns, and preferences to reuse across projects.
- **Embedded Editor**: Edit files in-place with full LSP support, including hover, definition, and refactoring.

## Getting Started

1. Install SoulForge globally: `npm install -g @proxysoul/soulforge`
2. Navigate to your project directory and run `soulforge` or `sf`.
3. Explore the available tools by pressing Ctrl+S or typing `/tools`.

## Configuration

SoulForge can be configured via a `soulforge.config.json` file in your project root. See the [configuration guide](https://github.com/proxysoul/soulforge/blob/main/docs/configuration.md) for details.
